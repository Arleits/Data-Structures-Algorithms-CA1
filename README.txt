Data Structures and Algorithms CA1

This application as according to the brief, uses 3 ADTs in its functionality; single linked list, queue and stack.
It works by adding stations to a single linked list using the add button and a textfield input that is used for all user input. A station
has its positioned displayed on the left in numerical order. You can remove or select a station by inputing the stations id.
To use other buttons below the textfield, a station must be selected first. You can queue or dequeue a bike and that information is stored in 
a queue or stack respectively.
Test data button exists to populate the linked list with a sample of stations for testing purposes.
Single linked list, queue and stack sample code/explanations was provided by Hamilton on moodle. See references for more information.
